Treasure Hunter

  • 프로젝트 설명: 2D 퍼즐 게임 - Tech 리드 / 3인팀
  • 기술 스택: C++, Doodle, Visual Studio
  • Github URL: Project Link

  • C++와 Doodle 라이브러리를 사용하여 맞춤형 엔진을 구축했습니다.

  • 적의 움직임을 위한 DFS을 사용한 길찾기 알고리즘을 구현했습니다.

  • 게임의 안정성을 높이기 위해 디버깅과 버그 수정에 집중했습니다

  • OpenGL을 사용하여 만든 Doodle 라이브러리를 통해 그래픽의 기초를 배우고 이를 실제 게임에 적용했습니다.

  • C++를 사용하여 적 AI, 게임 로직 및 기타 기능을 구현하고 성능 최적화에 중점을 두었습니다.

  • 2D 스프라이트를 사용하여 맞춤형 애니메이션 시스템을 구현했습니다.
